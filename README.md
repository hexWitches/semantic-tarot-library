# Semantic Tarot Library

**SMT (Semantic Tarot Library)** is a Semantic Digital Library designed to preserve and navigate the complex history of Tarot through **Linked Open Data (LOD)**.

Moving beyond traditional digital archives, SMT transforms historical decks into a machine-readable Knowledge Graph. By mapping the relationships between decks, cards, artists, and lineages, the library allows for multi-dimensional exploration: from a single card to its creator, to the creator’s entire body of work, and finally to the historical lineages that connect 15th-century masterpieces to modern iconography.

The site of the project is live at https://w3id.org/smt-library/.
