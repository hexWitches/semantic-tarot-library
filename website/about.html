About

The SMT Library is a digital ecosystem designed to preserve and navigate the complex history of Tarot through Linked Open Data (LOD). By aggregating data from different sources and modeling the intricate relationships between cards, decks, creators, and symbolic motifs, the project provides a structured environment for the exploration of the Tarot tradition. Rather than treating Tarot exclusively as a tool for divination, this project approaches tarot cards as significant cultural artifacts.

A central objective is the development of a semantic environment that moves beyond the simple collection of factual data to facilitate meaningful connections between entities. To achieve this, the project adopts Semantic Web principles in both its conceptual design and its technical implementation, functioning not only as a digital repository but as an interface for navigating interconnected knowledge structures.

The SMT Ontology

At the core of the library lies the SMT Ontology, a formal model designed to map the intricate landscape of the tarot domain and provide the conceptual foundation for the knowledge graph. The ontology imports the ODI ontology as the starting framework, re-using core entities such as odi:DeckCard, odi:TarotDeck, odi:Person. Thus, the SMT ontology functions as an extension, it defines fundamental entities – smt:MajorArcana, smt:MinorArcana, smt:Archetype, smt:SymbolicFigure – and provides a robust yet flexible structure for organizing data. Built on Semantic Web principles, it transforms isolated facts into a machine-readable network that enables deep exploration across historical traditions and artistic interpretations.

The SMT Graph serves as the operational framework for the web application and facilitates advanced data exploration through a dedicated SPARQL endpoint. The integration of a SPARQL endpoint directly connected to the knowledge graph supports advanced research by allowing users to query the dataset directly from the browser. This configuration enables investigations into entity relationships that may go beyond the functionalities offered by the graphical interface. By exposing the dataset in this manner, the project aligns with the core principles of LOD, ensuring that the structured information remains interoperable and accessible to external systems. In this way, the ontology and knowledge graph function not just as back-end structures, but as openly accessible components of a broader digital library ecosystem.

The Web Application

The web application provides intuitive access to the underlying data while offering multiple strategies for discovery. By combining search, filtering, and semantic linking, the platform enables both exploratory browsing and structured, research-oriented interaction.

A key feature of the interface is the integration of links that reflect the relationships modeled within the knowledge graph, allowing users to move seamlessly across related entities. Each resource page includes dedicated sections for related items, such as connections between specific cards and their creators or thematic elements, which are visualized through components that highlight the underlying conceptual network. The Explore page further expands this view by examining the systemic connections between decks and the individuals who brought them to life, utilizing in-depth thematic insights to redirect users toward related topics within the library.

The collection is accessible through multiple entry points, allowing users to browse by decks, individual cards, or through contextual materials that situate these objects within their broader cultural and symbolic frameworks. This approach encourages users to move beyond isolated resources to discover the network of relationships that define the domain.

To support targeted retrieval, the system employs a client-side engine that features keyword matching, fuzzy matching, and auto-completion suggestions. As users type in the search bar, the interface proposes entities based on keywords linked directly to the dataset. These search functionalities are complemented by specific filters that allow users to refine exploration by specific attributes, supporting the discovery of unexpected relationships within the data.

Bibliography and Sitography

The description of the cards, decks and archetypes and the thematic insights and explanatory materials take inspiration from the following sources.

  • Le pèlerinage des bateleurs: Histoire, légendes et commentaires sur les 22 arcanes majeurs du tarot de Marseille, Jean-Claude Flornoy, 2007
  • The Game of Saturn: Decoding the Sola-Busca Tarocchi, Peter Mark Adams, 2007
  • The Pictorial Key To The Tarot: A Visual Companion to the Rider Waite Tarot, Arthur Edward Waite, 1910
  • The Visconti-Sforza Tarot: Traveling through Medieval Myths to the Art of the Renaissance, Midirath Arianrhod and Max Kahl, 2023
  • A History of tarot cards
  • Before Fortune-Telling: The History and Structure of Tarot Cards
  • Mazzi Visconti-Sforza
  • Tarot cards: the pocket-sized Renaissance
  • The Origin of Tarot Cards

The Team

About the project

The SMT Library is designed to preserve and navigate the complex iconography and history of Tarot through Linked Open Data (LOD). This project was developed for the Semantic Digital Libraries course (a.y. 2025/2026) within the Digital Humanities and Digital Knowledge Master's Degree at the University of Bologna.
